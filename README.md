
# Student Management System

A simple Student Management System built using Python. This project allows users to add, view, and search student records. Data is stored in a text file using JSON format.

## Features

✅ Add Student Records  
✅ View All Students  
✅ Search Student by Name  
✅ Data Storage using JSON  
✅ User-Friendly Menu System

## Technologies Used

- Python 3
- JSON Module
- File Handling

## Project Structure

```text
Student-Management-System/
│
├── main.py
├── students.txt
└── README.md
```

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Run:

```bash
python main.py
```

## Menu

1. Add Student
2. View Students
3. Search Student
4. Exit
